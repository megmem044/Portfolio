Key Features

Shift-Based Management

Volunteers browse and apply for specific event shifts. Coordinators review and approve/decline requests.

Hours Tracking

Automatic calculation of volunteer hours worked based on shift start and end times.

Event Ratings

Volunteers rate events on a 5-star scale with optional comments. Coordinators view all feedback.

Volunteer Profiles

Complete volunteer management with profile editing, certificate uploads, and preference management.

Event Dashboard

Coordinators get detailed event dashboards showing volunteers, shifts, ratings, and schedules.

API-Driven

13 RESTful API endpoints for seamless frontend integration and real-time data sync.
